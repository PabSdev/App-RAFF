RASFF Alerts Extractor
Aplicación en Python extrae alertas de seguridad alimentaria desde la web de RASFF y las guarda en un archivo Excel. Usa Selenium para la automatización web, Pandas para el manejo de datos y Flet para la interfaz gráfica.

CARACTERÍSTICAS

Automatización con Selenium: Extrae datos de alertas de la web de RASFF.
Almacenamiento en Excel: Guarda las alertas en un archivo con historial.
Interfaz gráfica moderna: Uso de Flet para facilitar la interacción con los usuarios.

INSTALACIÓN Y CONFIGURACIÓN

Clonar el repositorio git clone https://github.com/tuusuario/rasff-alerts-extractor.git cd rasff-alerts-extractor
Instalar dependencias pip install -r requirements.txt
Ejecutar la aplicación python app.py

USO

Selecciona una fecha: Para obtener las alertas de un día específico.
Selecciona el archivo Excel: Donde se guardarán las alertas.
Haz clic en "Extraer y Guardar": La aplicación obtendrá las alertas y las almacenará en el archivo seleccionado.

REQUISITOS

Python 3.8+
Google Chrome instalado
ChromeDriver compatible con tu versión de Chrome
